# Устранение проблем подключения

## Проблема: "Ошибка подключения к серверу"

### Решение 1: Проверьте настройки URL в `app_config.dart`

Откройте файл `seller/lib/config/app_config.dart` и убедитесь, что используется правильный URL:

**Для Android эмулятора:**
```dart
static const String baseUrl = 'http://10.0.2.2:3000';
```

**Для реального Android устройства:**
1. Узнайте IP адрес вашего компьютера:
   - Windows: `ipconfig` в командной строке
   - Mac/Linux: `ifconfig` или `ip addr`
2. Найдите IP адрес в локальной сети (обычно начинается с 192.168.x.x)
3. Измените в `app_config.dart`:
```dart
static const String baseUrl = 'http://192.168.1.100:3000'; // замените на ваш IP
```

**Для iOS симулятора:**
```dart
static const String baseUrl = 'http://localhost:3000';
```

### Решение 2: Убедитесь, что бэкенд запущен

```bash
cd backend_nestjs
npm run start:dev
```

Проверьте, что бэкенд запущен на порту 3000:
```bash
netstat -ano | findstr :3000
```

### Решение 3: Проверьте файрвол

Убедитесь, что файрвол Windows не блокирует порт 3000.

### Решение 4: Для реального устройства

1. Убедитесь, что телефон и компьютер в одной Wi-Fi сети
2. Используйте IP адрес компьютера, а не localhost
3. Проверьте, что бэкенд слушает на всех интерфейсах (0.0.0.0), а не только localhost

### Решение 5: Перезапустите приложение

После изменения `app_config.dart`:
```bash
cd seller
flutter clean
flutter pub get
flutter run
```
