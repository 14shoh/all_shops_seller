# Seller App - Мобильное приложение для продавцов

Flutter приложение для продавцов магазинов.

## Установка зависимостей

```bash
flutter pub get
```

## Установка иконки приложения

Для установки иконки сканера штрих-кода выполните следующие шаги:

1. **Создайте PNG иконку** (1024x1024 пикселей):
   - Откройте `assets/icon_scanner.html` в браузере (иконка автоматически скачается)
   - Или используйте онлайн конвертер: https://convertio.co/svg-png/
   - Загрузите `assets/icon_scanner.svg` и скачайте PNG размером 1024x1024
   - Сохраните как `assets/icon_scanner.png`

2. **Сгенерируйте иконки для платформ**:
   ```bash
   flutter pub run flutter_launcher_icons
   ```

3. **Пересоберите приложение**:
   ```bash
   flutter clean
   flutter pub get
   flutter run
   ```

Подробная инструкция в файле `GENERATE_ICON.md`.

## Настройка

1. Откройте файл `lib/config/app_config.dart`
2. Измените `baseUrl` на адрес вашего бэкенда:
   - Для эмулятора Android: `http://10.0.2.2:3000`
   - Для реального устройства: `http://IP_ВАШЕГО_КОМПЬЮТЕРА:3000`
   - Для iOS симулятора: `http://localhost:3000`

### Что такое Hot Reload?

Hot Reload позволяет видеть изменения в коде **мгновенно** без перезапуска приложения!

**Как использовать:**
1. Запустите приложение через `flutter run` или скрипт выше
2. Внесите изменения в код
3. **Сохраните файл (Ctrl+S)** - изменения применятся автоматически!
4. Или нажмите `r` в терминале для Hot Reload
5. Нажмите `R` для Hot Restart (если Hot Reload не помог)
6. Нажмите `q` для выхода

**Подробнее:** См. [HOT_RELOAD_GUIDE.md](HOT_RELOAD_GUIDE.md)

## Структура проекта

```
lib/
├── config/           # Конфигурация приложения
├── core/             # Основные компоненты
│   ├── constants/    # Константы
│   ├── models/       # Модели данных
│   ├── providers/    # State management (Provider)
│   ├── router/       # Навигация (GoRouter)
│   └── services/     # API и Storage сервисы
└── features/         # Функциональные модули
    ├── auth/         # Аутентификация
    ├── home/         # Главный экран
    ├── products/     # Управление товарами
    ├── sales/        # Продажи
    └── reports/      # Отчеты
```
