# Инструкция по тестированию Seller App

## Подготовка

1. **Убедитесь, что бэкенд запущен:**
```bash
cd backend_nestjs
npm run start:dev
```

2. **Настройте API URL:**
   - Откройте `lib/config/app_config.dart`
   - Для эмулятора Android: `http://10.0.2.2:3000`
   - Для реального устройства: `http://ВАШ_IP:3000`
   - Для iOS симулятора: `http://localhost:3000`

3. **Установите зависимости:**
```bash
cd seller
flutter pub get
```

## Сценарии тестирования

### 1. Авторизация
- Откройте приложение
- Введите логин и пароль одного из продавцов
- Проверьте переход на главный экран

### 2. Добавление товара
- Нажмите "Добавить товар"
- Заполните форму:
  - Название товара
  - Штрихкод (можно отсканировать)
  - Категория
  - Закупочная цена
  - Количество
- Для магазина одежды: выберите размер
- Для продуктового магазина: укажите вес
- Сохраните товар

### 3. Создание продажи
- Нажмите "Новая продажа"
- Отсканируйте штрихкод товара
- Введите количество и цену продажи
- Добавьте еще товары в чек
- Проверьте автоматический расчет суммы
- Оформите продажу

### 4. Просмотр отчетов
- Перейдите в "Отчеты"
- Выберите дату
- Проверьте отображение:
  - Общей суммы продаж за день
  - Количества продаж
  - Списка проданных товаров

## Возможные проблемы

### Ошибка подключения к API
- Проверьте, что бэкенд запущен
- Проверьте правильность URL в `app_config.dart`
- Для реального устройства убедитесь, что телефон и компьютер в одной сети

### Ошибка авторизации
- Проверьте правильность логина и пароля
- Убедитесь, что пользователь активен в базе данных

### Камера не работает
- Проверьте разрешения приложения на доступ к камере
- Для Android: настройки приложения → Разрешения → Камера
